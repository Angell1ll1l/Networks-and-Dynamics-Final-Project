# Data Instructions

This repository does not require downloading any private or external dataset.

All results are generated by simulation from the four-oscillator model.

## How data is produced

- Run scripts from `scripts/`.
- Each script simulates oscillator trajectories and computes lag-correlation or synchronization metrics.
- Figures are saved to `figures/`.

## Optional storage policy

If you choose to save intermediate `.mat` files during experiments, store them in this `data/` directory and keep generated binaries out of version control unless needed for replication evidence.
